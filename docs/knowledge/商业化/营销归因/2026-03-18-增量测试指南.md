## 核心摘要

增量测试为营销人员提供**精度和控制**，专注于因果提升，测量广告真正增加了什么，而不是应用本身的表现。

### 三大关键行动

1. **实施增量测试**：使用Singular Incrementality建立清晰的因果关系，比较控制组和测试组
2. **增强营销活动透明度**：利用透明结果向利益相关者传达营销活动影响
3. **简化A/B测试流程**：采用简单的A/B保留测试快速实施和扩展

---

## 问题：证明渠道真正增量价值的挑战

移动营销人员今天面临巨大压力：
- 需要交付更好的绩效
- 证明支出的合理性
- 在多个渠道高效扩展
- 避免预算浪费和蚕食自然增长

**核心问题**：
- 重定向是否值得预算和时间？
- 媒体组合策略是否优化？
- 重定向营销活动是否提供增量价值？
- 营销活动是否蚕食了本来就会参与的用户？
- 重定向策略是否尽可能高效？

## Singular Incrementality：真实受众的真实提升

### 工作原理

基于Singular强大的Audiences解决方案进行精确A/B测试：

```
1. Singular创建受众细分
2. Singular与合作伙伴分享用于重定向
3. Singular Incrementality：
   - 将受众分成控制组（无处理）和测试组（展示广告）
   - 按常规运行营销活动
   - 测试任意时长
   - 比较各组间的转化或关键指标
   - 差异（增量提升）= 广告的附加价值
```

### 核心价值

**精度**：提供干净、有效的广告影响测量

**关键洞察**：现在可以看到营销活动是推动净新转化，还是仅仅转移现有或自然用户。

---

## Singular Incrementality测试的价值

### 1. 更少假设的直接因果关系

使用真实控制组，比较看到和没看到广告的用户的实际行为，提供营销活动绩效的强有效性。

### 2. 清晰、透明的结果

**控制组**的含义对所有人都很清楚：这些是与目标受众一样的真实用户，只是没有看到你的广告。

**价值**：向领导层或财务团队解释结果时非常有帮助。

### 3. 更简单的测试和分析

**A/B保留测试更简单**：分割、运行、比较。快速实施，简单重复。

**好处**：帮助你在营销活动和团队间扩展增量测试。

### 4. 证明重定向策略的价值

**Beta测试中的关键发现**：

| 发现类型 | 结果 |
|----------|------|
| 测试者A | 看到显著增量提升，决定投资更多重定向（之前认为不可行） |
| 测试者B | 验证了重定向渠道没有提供所需增量，转向更相关的ROI正向渠道 |

**没有Singular Incrementality的洞察**：
- 一个会失去宝贵的增量提升
- 另一个会继续在非绩效上浪费时间和支出

## 实践应用

### 何时使用增量测试

- 评估重定向营销活动的真正价值
- 验证媒体组合策略
- 检测渠道蚕食效应
- 优化预算分配决策

### 测试流程

```
┌─────────────────────────────────────────────────────┐
│              增量测试流程                            │
├─────────────────────────────────────────────────────┤
│  1. 定义测试目标                                     │
│     ↓                                               │
│  2. 创建受众细分                                     │
│     ↓                                               │
│  3. 分配控制组/测试组                                │
│     ↓                                               │
│  4. 运行营销活动                                     │
│     ↓                                               │
│  5. 比较转化和关键指标                               │
│     ↓                                               │
│  6. 分析增量提升                                     │
│     ↓                                               │
│  7. 做出数据驱动的决策                               │
└─────────────────────────────────────────────────────┘
```

## 关键概念

| 术语 | 定义 |
|------|------|
| 增量提升 | 广告真正增加的价值，而非自然发生的转化 |
| 控制组 | 与目标受众相同但不展示广告的真实用户 |
| 测试组 | 展示广告的目标用户 |
| 因果关系 | 直接证明广告导致的转化，而非相关性 |
| 蚕食效应 | 广告获取本会自然转化的用户 |
