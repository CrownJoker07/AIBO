# Tenjin MCP Server: 移动UA数据直接查询

> 移动广告ROI与归因分析核心知识点
> 来源: https://tenjin.com/blog/tenjins-mcp-server-query-your-data-directly/
> 获取日期: 2026-03-18

---

## 概述

Tenjin的MCP Server允许直接查询移动UA数据，获取CPI、ROAS、留存等关键指标的洞察。它将花费、归因、收入和队列数据统一在一个整体视图中。

---

## 什么是MCP Server?

### 定义
- **MCP (Model Context Protocol)**: 模型上下文协议
- **核心功能**: 允许AI模型直接查询Tenjin数据

### 核心价值
| 价值点 | 描述 |
|--------|------|
| **直接查询** | 无需手动导出数据 |
| **统一视图** | 花费+归因+收入+队列 |
| **实时洞察** | 快速获取关键指标 |

---

## 可查询的核心指标

### 用户获取指标
| 指标 | 描述 | 用途 |
|------|------|------|
| **CPI** | 每安装成本 | 评估获客效率 |
| **Installs** | 安装数量 | 规模评估 |
| **Impressions** | 展示数量 | 覆盖评估 |
| **Clicks** | 点击数量 | 兴趣评估 |

### 收入与ROI指标
| 指标 | 描述 | 用途 |
|------|------|------|
| **ROAS** | 广告支出回报率 | 评估整体效率 |
| **Revenue** | 收入 | 评估变现效果 |
| **LTV** | 用户生命周期价值 | 长期价值评估 |

### 留存与参与指标
| 指标 | 描述 | 用途 |
|------|------|------|
| **Retention** | 留存率 | 用户粘性评估 |
| **DAU/MAU** | 日/月活跃用户 | 规模与活跃度 |
| **Sessions** | 会话数 | 用户参与度 |

## 技术实现

### 查询示例
```
查询: "过去7天Google Ads的ROAS是多少?"
→ MCP Server返回实时数据
→ 无需手动导出/分析
```

### 数据更新频率
- 花费数据: 每日更新
- 归因数据: T+2到T+3
- 收入数据: T+3 (含广告收入延迟)
