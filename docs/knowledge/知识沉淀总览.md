## 核心指标速查

### 游戏运营基准

| 指标 | 休闲游戏 | 超休闲游戏 |
|------|----------|------------|
| D1 留存 | 25-35% | 20-25% |
| D7 留存 | 12-18% | 8-12% |
| D30 留存 | 6-10% | 3-6% |
| CPI | $0.5-2.5 | 越低越好 |
| 付费转化率 | 2-5% | - |
| 激励视频 eCPM | $15-40 | - |

### 投放决策阈值

| 指标 | 阈值 | 行动 |
|------|------|------|
| 7日ROAS | < 60% | 暂停/降预算 |
| 7日ROAS | 80-100% | 维持投放 |
| 7日ROAS | > 100% | 加投扩量 |
| 填充率 | < 80% | 增加广告平台 |
| eCPM | < $10 | 优化广告位 |
