## 二、核心分析模型

### 2.1 RFM 客户细分模型

**来源**: [CleverTap: RFM Analysis](https://clevertap.com/blog/rfm-analysis/)

#### 三维评分体系

```
RFM评分 = Recency(1-5) × Frequency(1-5) × Monetary(1-5)
       → 最多125个独特客户细分
```

#### 核心细分策略矩阵

| 细分 | 特征 | 策略 |
|------|------|------|
| Champions | R↑F↑M↑ | 奖励忠诚，新品体验 |
| At Risk | R↓F↑M↑ | 强力召回 |
| Can't Lose Them | R↓F↑M↑ | 高端挽留 |
| New Customers | R↑F低M中 | 欢迎序列 |
| Hibernating | R↓F低M低 | 最后激活尝试 |

#### RFM 效果基准

| 指标 | RFM驱动 | 非定向 | 提升幅度 |
|------|---------|--------|----------|
| 点击率 (CTR) | +50% | 基准 | 50% |
| 留存率 | +10-20% | 基准 | 10-20% |
| ROI | +77% | 基准 | 77% |

### 2.2 Cohort 分析框架

**来源**: [CleverTap: Cohort Analysis](https://clevertap.com/blog/cohort-analysis/)

```
┌─────────────────────────────────────────────────────┐
│              Cohort 分析类型矩阵                     │
├─────────────────────────────────────────────────────┤
│                                                     │
│  获取Cohort    →    何时流失?    →    评估Onboarding│
│  行为Cohort    →    为何流失?    →    优化体验      │
│  时间Cohort    →    何时效果好?  →    活动评估      │
│  细分Cohort    →    谁更忠诚?    →    个性化策略    │
│  规模Cohort    →    能否扩展?    →    可扩展性测试  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**关键洞察**: 购物车放弃者意图在48小时内快速消失

```
行为Cohort对比:
完成交易 → Day1: 43% → Day7: 18%
放弃购物 → Day1: 4%  → Day7: 0%

策略: 放弃者需24小时内紧急召回
```

## 三、关键指标体系

### 3.1 14项客户留存指标

**来源**: [CleverTap: Retention Metrics](https://clevertap.com/blog/customer-retention-metrics/)

| 类别 | 指标 | 公式速记 |
|------|------|----------|
| **核心留存** | 留存率 | (期末-新增)/期初 |
| | 流失率 | 流失/期初 |
| | 收入流失 | 流失MRR/期初MRR |
| **价值** | AOV | 收入/订单数 |
| | CLV | 价值×频率×周期 |
| **满意度** | NPS | 推荐者%-贬损者% |
| | CSAT | 正面/总数 |
| **健康度** | 健康评分 | Σ(行动价值×权重) |
| | MRR | 账户数×ARPU |
| **参与度** | 粘性 | DAU/MAU |
| | CES | 评分和/回应数 |
| **行为** | 购买频率 | 订单数/客户数 |
| | 退货率 | 退货/售出 |

### 3.2 领先 vs 滞后指标

**来源**: [CleverTap: Leading vs Lagging Indicators](https://clevertap.com/blog/leading-vs-lagging-indicators/)

| 类型 | 特点 | 示例 |
|------|------|------|
| **领先指标** | 预测性、前瞻性 | 试用数、新注册、页面浏览 |
| **滞后指标** | 结果性、回顾性 | 收入、流失率、利润 |

```
领先指标回答: "我们将去向哪里?"
滞后指标回答: "我们做得怎么样?"
```

### 3.3 DAU/MAU 应用粘性

```
DAU/MAU 比率 = 日活跃用户 / 月活跃用户

粘性基准:
┌─────────────────────────────────────┐
│  > 50%  → 超高粘性 (社交/通讯)      │
│  20-50% → 高粘性 (工具/娱乐)        │
│  10-20% → 中等粘性 (电商/内容)      │
│  < 10%  → 低粘性 (低频需求)         │
└─────────────────────────────────────┘
```

### 4.3 Drop-Off Rate 分析

```
Drop-Off Rate = (进入用户数 - 完成用户数) / 进入用户数 × 100%

示例 - 注册漏斗:
┌─────────────────────────────────────┐
│ 访问注册页    1000人                │
│     ↓ Drop-off: 30%                 │
│ 填写表单      700人                 │
│     ↓ Drop-off: 50%                 │
│ 提交表单      350人                 │
│     ↓ Drop-off: 20%                 │
│ 验证邮箱      280人                 │
│     ↓ Drop-off: 10%                 │
│ 完成注册      252人                 │
└─────────────────────────────────────┘

总转化率: 25.2%
最大流失点: 填写表单 (50%)
优化优先级: 表单简化
```

## 五、客户生命周期价值 (CLV)

### 5.1 CLV 计算公式

```
基础: CLV = APV × PF × 生命周期
SaaS: CLV = (ARPU × 毛利率) / 流失率
含利润: CLV(利润) = CLV(收入) × 毛利率

黄金法则: CLV:CAC = 3:1
```

### 5.2 CLV 三种模型对比

| 模型 | 特点 | 应用场景 |
|------|------|----------|
| **历史CLV** | 回顾性，已实现价值 | 客户细分，Top25%识别 |
| **预测CLV** | 前瞻性，需要假设 | 营销预算分配，高价值客户早期识别 |
| **分段CLV** | 群体平均值 | 渠道价值对比，推荐来源分析 |

### 5.3 CLV 提升 14 策速查

1. 忠诚计划 → 2. 优化Onboarding → 3. 卓越客服 → 4. 个性化触点 → 5. 再营销活动
6. 简化UX → 7. 推荐激励 → 8. 客户教育 → 9. 全渠道参与 → 10. 智能向上/交叉销售
11. 识别VIP → 12. 反馈行动 → 13. 品牌社区 → 14. 持续改进

## 七、流失分析框架

### 7.1 客户流失原因

| 流失原因 | 占比 | 预防策略 |
|----------|------|----------|
| **产品体验差** | 32% | UX优化、Onboarding改进 |
| **客服不力** | 24% | 多渠道支持、快速响应 |
| **价格敏感** | 18% | 价值证明、灵活定价 |
| **缺乏参与** | 15% | 个性化互动、忠诚计划 |
| **竞品吸引** | 11% | 差异化、持续创新 |

### 7.2 流失率计算

```
客户流失率 = (期初客户 - 期末客户 + 新增客户) / 期初客户 × 100%

收入流失率 = 流失MRR / 期初MRR × 100%

负流失 (Negative Churn):
当 扩展收入 > 流失收入 时实现
= SaaS企业的黄金目标
```
