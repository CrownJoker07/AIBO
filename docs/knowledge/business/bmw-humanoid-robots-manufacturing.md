# BMW 人形机器人在欧洲制造业的首次部署

## 元数据

- **来源**: https://www.artificialintelligence-news.com/news/bmw-humanoid-robots-manufacturing-europe-leipzig/
- **作者**: Dashveenjit Kaur
- **发布日期**: 2026-03-13
- **获取日期**: 2026-03-17
- **分类**: 制造业AI、物理AI、人形机器人

## 核心要点

**BMW Group** 在德国 **Leipzig 工厂**部署人形机器人，这是欧洲制造业首次人形机器人试点。使用 **Hexagon Robotics** 的 **AEON** 轮式人形机器人进行电池组装，标志着物理 AI 不再只是北美和东亚的故事。

## AEON 机器人规格

### 设计哲学

> "我们不是做跳舞的生意——我们做的是工作的生意。"
> — Arnaud Robert, Hexagon Robotics 总裁

### 技术参数

| 参数 | 规格 |
|------|------|
| 身高 | 1.65 米 |
| 重量 | 60 公斤 |
| 移动方式 | 轮式（工厂地面效率更高） |
| 最高速度 | 2.5 米/秒 |
| 电池更换 | 23 秒自主更换 |
| 传感器数量 | 22 个集成传感器 |

### 传感器配置

- 周边摄像头
- 飞行时间传感器
- 红外传感器
- SLAM 摄像头
- 麦克风
- **360 度实时空间感知**
- 质量检测能力（传统固定机器人无法实现）

## Leipzig 部署计划

### 时间表

| 阶段 | 时间 | 内容 |
|------|------|------|
| 首次测试 | 2025年12月 | AEON 首次测试部署 |
| 进一步测试 | 2026年4月 | 第二轮测试 |
| 全面试点 | 2026年夏季 | 2 台 AEON 同时工作 |

### 应用场景

1. **高压电池组装**
2. **外部零件组件制造**

### Leipzig 工厂选择原因

- BMW 德国技术最全面的工厂
- 集成：电池生产 + 注塑 + 冲压车间 + 车身车间 + 最终组装
- 成功部署 = 全生产光谱验证
