# 超休闲游戏设计与用户获取

## 概述

超休闲游戏（Hyper-Casual）以极简玩法和快速迭代为特点，是移动游戏市场的重要品类。

## 游戏设计最佳实践

### 核心玩法

- **极简操作**：单点触控或滑动
- **即时满足**：玩家数秒内理解游戏
- **短循环**：单局 30-60 秒
- **反馈设计**：视觉、音效、震动反馈

### 进度与留存

- 清晰的里程碑系统
- 奖励性进度（解锁、皮肤、道具）
- 创造"再来一局"的冲动
- 难度平衡：有挑战但不挫败

### 变现设计

- **激励视频广告**：用户主动观看，获得奖励
- **插屏广告**：关卡间隙展示
- **避免打断**：不在游戏进行中展示广告
- **混合模式**：IAP + 广告（混合休闲）

## 用户获取策略

### 素材测试

- 测试多种广告素材
- 可玩广告（Playable Ads）效果好
- 前 3 秒抓住注意力
- 展示令人满足的游戏时刻
- 持续 A/B 测试

### 关键指标

| 指标 | 说明 | 目标范围 |
|------|------|----------|
| CPI | 每安装成本 | 越低越好 |
| Day 1 留存 | 次日留存率 | >25% |
| Day 7 留存 | 7日留存率 | >10% |
| LTV | 用户生命周期价值 | >CPI |
| ROAS | 广告支出回报率 | >100% |

### 投放平台

- Facebook/Meta Ads
- Google App Campaigns
- TikTok（超休闲用户群体）
- Unity Ads / ironSource

## 开发建议

1. 快速原型验证核心玩法
2. 软发布测试市场反应
3. 数据驱动迭代优化
4. 规模化前确保 LTV > CPI
