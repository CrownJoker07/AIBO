## 一、休闲游戏 vs 中重度游戏：核心差异

| 维度 | 休闲游戏 | 中重度游戏 |
|------|----------|------------|
| **变现模式** | 广告(IAA)为主 + 内购(IAP)为辅 | 内购(IAP)为主 + 广告为辅 |
| **用户规模** | 大规模获客，追求爆量 | 精准获客，追求质量 |
| **CPI区间** | $0.5-3（低价竞争） | $3-20+ |
| **ARPU** | 低（靠广告填充） | 高（靠大R付费） |
| **LTV回收周期** | 短（7-14天见分晓） | 长（30-90天） |
| **核心指标** | eCPM、填充率、留存 | 付费率、ARPU、LTV |

## 二、休闲游戏核心指标体系

### 1. 广告变现指标（IAA）

| 指标 | 英文 | 定义 | 优化方向 |
|------|------|------|----------|
| **eCPM** | effective Cost Per Mille | 千次展示收入 | 广告场景设计、填充率 |
| **填充率** | Fill Rate | 请求成功展示/总请求 | 聚合平台配置 |
| **展示率** | Show Rate | 实际展示/广告请求 | 用户体验设计 |
| **点击率** | Ad CTR | 广告点击/展示 | 素材样式、广告位设计 |
| **广告ARPU** | Ad ARPU | 广告收入/DAU | 场景数量、ecpm提升 |

**休闲游戏 eCPM 参考区间（美元）：**
```
激励视频: $15-40
插屏: $5-15
原生广告: $8-20
banner: $1-3
```

### 2. 用户质量指标

| 指标 | 定义 | 休闲游戏健康值 |
|------|------|----------------|
| **次留 (D1)** | 次日留存率 | 40-55% |
| **7留 (D7)** | 7日留存率 | 15-25% |
| **30留 (D30)** | 30日留存率 | 5-10% |
| **平均游戏时长** | 用户单次游戏时长 | 3-8分钟 |
| **人均广告曝光** | 每人每日看广告次数 | 5-15次 |

**留存曲线解读：**
```
休闲游戏典型曲线（陡峭下降）：
D1: 45% → D3: 20% → D7: 12% → D14: 8% → D30: 5%

健康信号：
- D1 > 45%，说明吸量且上手快
- D1-D3 衰减 < 50%，说明核心玩法有粘性
- D7 > 15%，说明有长线用户
```

### 3. 获客指标

| 指标 | 休闲游戏目标值 |
|------|----------------|
| **CPI** | $0.5-2.5 |
| **CTR** | 2-5% |
| **CVR** | 15-30% |
| **ROAS（7日）** | > 80% |
| **ROAS（14日）** | > 100% |

## 三、休闲游戏数据分析框架

### 1. 渠道效果评估矩阵

```
                    高回收
                      ↑
                      │
         明星渠道     │    潜力渠道
         (加投扩量)   │    (优化提升)
                      │
    ──────────────────┼─────────────────→ 高量级
                      │
         问题渠道    │    流量渠道
         (暂停/优化) │    (维持现状)
                      ↓
                    低回收

维度说明：
- X轴：量级（新增用户数）
- Y轴：回收（LTV/CPI 或 ROAS）
```

### 2. 核心分析维度

#### 渠道维度
- 各渠道新增用户数
- 各渠道CPI
- 各渠道7日/14日ROAS
- 各渠道填充率、eCPM
- 各渠道留存表现

#### 素材维度
- 视频 vs 图片素材表现
- 不同玩法展示素材效果
- 真人素材 vs 动画素材
- 福利向 vs 玩法向素材

#### 人群定向
- 年龄段分布（休闲游戏18-35为主）
- 地域分布（优先投放高eCPM地区）
- 兴趣标签效果

## 四、休闲游戏关键决策阈值

### 投放决策

| 指标 | 阈值 | 行动 |
|------|------|------|
| **7日ROAS** | < 60% | 立即暂停/大幅降预算 |
| **7日ROAS** | 60-80% | 保持观察，优化出价 |
| **7日ROAS** | 80-100% | 维持投放 |
| **7日ROAS** | > 100% | 加投扩量 |
| **CPI** | 超过目标30% | 换素材/收窄定向 |
| **次留** | < 35% | 检查产品问题或渠道匹配度 |

### 变现决策

| 指标 | 阈值 | 行动 |
|------|------|------|
| **填充率** | < 80% | 更换/增加广告平台 |
| **eCPM** | < $10 | 优化广告位/增加激励视频 |
| **人均广告曝光** | < 3次/天 | 增加广告触发场景 |
| **激励视频点击率** | < 10% | 优化奖励力度/广告样式 |

## 六、休闲游戏优化策略

### 1. 获客优化

**CPI 优化：**
- 测试多种素材风格（休闲/解压/爽感）
- 突出"上手快"、"无需联网"等卖点
- 人群定向：18-35岁，兴趣标签"小游戏"、"休闲游戏"

**CTR 优化：**
- 视频素材：前3秒要抓住注意力
- 展示真实游戏画面比展示代言人更好
- 福利向素材（钻石/金币）点击率高

### 2. 变现优化

**提升 eCPM：**
- 激励视频 > 插屏 > 原生 > banner
- 多家广告平台聚合（Max、Unity Ads、AppLovin）
- 优化广告触发时机（关卡结束/升级/体力用完）

**提升填充率：**
- 主流平台都接上（Google AdMob、Meta、Mintegral）
- 避免只接一家导致填充不满
- 针对不同地区选择不同平台

### 3. 留存优化

**休闲游戏留存杀手：**
- 广告太多影响体验
- 前期引导太长
- 玩法太难容易劝退

**优化方向：**
- 前3关要足够简单有趣
- 早期不要插入广告
- 流失节点分析（D3/D7重点关注）

## 八、休闲游戏关键公式

```
广告收入 = DAU × 人均广告曝光次数 × eCPM / 1000
综合LTV = 广告LTV + IAP LTV
回收周期 = CPI / (日ARPU × 回收率)
ROAS = 广告收入 / 广告花费 × 100%
盈亏平衡ROAS = 100% / 毛利率
```

**示例：**
```
CPI = $1.5
日均广告ARPU = $0.15
目标回收天数 = 14天

盈亏平衡ROAS = 100% (假设毛利率100%)
14天累计ARPU = $0.15 × 14 = $2.1
14日ROAS = $2.1 / $1.5 = 140% ✓
```
