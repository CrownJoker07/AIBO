# 游戏广告投放数据分析指南

> 来源: AIBO 知识库整理
> 分类: Business - 广告投放
> 更新: 2026-03-16

---

## 一、核心指标体系

### 1. 投放成本指标

| 指标 | 英文 | 公式/定义 | 优化方向 |
|------|------|-----------|----------|
| **CPM** | Cost Per Mille | 展示1000次的成本 | 素材优化、出价调整 |
| **CPI** | Cost Per Install | 每安装成本 = 花费/新增安装 | 素材吸引力、投放人群 |
| **CPC** | Cost Per Click | 每次点击成本 = 花费/点击数 | 素材优化、落地页优化 |
| **CPT** | Cost Per Time | 按时间计费成本 | 时段优化 |

### 2. 效果收益指标

| 指标 | 英文 | 公式/定义 | 核心价值 |
|------|------|-----------|----------|
| **ROI** | Return On Investment | 收入/投入×100% | 整体盈利能力 |
| **ROAS** | Return On Ad Spend | 广告收入/广告花费×100% | 广告投放效率 |
| **LTV** | Life Time Value | 用户生命周期总价值 | 用户质量评估 |
| **ARPU** | Average Revenue Per User | 总收入/总用户数 | 单用户价值 |
| **ARPPU** | Average Revenue Per Paying User | 总收入/付费用户数 | 付费用户质量 |

**关键关系：**
```
ROAS > 100% → 广告盈利
ROAS > 200% → 优质广告计划
LTV > CPI × 3 → 用户获取可持续
```

### 3. 转化漏斗指标

```
曝光 → 点击 → 下载 → 激活 → 注册 → 付费
```

| 指标 | 定义 | 优化重点 |
|------|------|----------|
| **CTR** | Click Through Rate = 点击/曝光 | 素材吸引力 |
| **CVR** | Conversion Rate = 转化/点击 | 落地页/商店页 |
| **激活率** | 激活/下载 | 安装包大小、闪退 |
| **注册率** | 注册/激活 | 注册流程 |
| **付费率** | 付费/激活 | 付费点设计 |

## 二、数据分析方法

### 1. 渠道维度分析

**核心问题：哪些渠道值得投？**

| 维度 | 分析内容 | 决策依据 |
|------|----------|----------|
| 量级 | 各渠道新增用户数 | 优先大容量渠道 |
| 成本 | 各渠道CPI/CPM | 选择低成本渠道 |
| 质量 | 各渠道LTV/付费率 | 选择高价值渠道 |
| 回收 | 各渠道ROAS | 选择高回收渠道 |

**渠道优先级矩阵：**
```
高量级 + 高回收 → 加投
高量级 + 低回收 → 优化素材/出价
低量级 + 高回收 → 扩展投放
低量级 + 低回收 → 暂停/替换
```

### 2. 时间维度分析

| 分析维度 | 应用场景 |
|----------|----------|
| **小时数据** | 找出高效投放时段，优化出价策略 |
| **日数据** | 追踪趋势，判断活动效果 |
| **周数据** | 排除周末波动，看长期趋势 |
| **生命周期** | 分析用户从注册到流失的全过程 |

### 3. 用户分群分析

| 分群方式 | 分析价值 |
|----------|----------|
| 来源渠道 | 区分用户质量 |
| 首次付费时间 | 识别付费行为模式 |
| 付费金额 | 识别高价值用户 |
| 留存天数 | 分析流失节点 |

## 三、优化策略

### 1. 出价优化

| 场景 | 策略 |
|------|------|
| CPI过高 | 降低出价/优化素材/收窄人群 |
| ROAS低于目标 | 降低出价/优化落地页 |
| 量级不足 | 提高出价/放宽定向/增加素材 |
| 成本稳定但想扩量 | 阶梯式提价(5-10%) |

### 2. 素材优化

**CTR 低时的优化方向：**
- 更换素材风格（写实vs卡通）
- 突出核心卖点（福利/玩法/剧情）
- 测试不同素材类型（视频>图片）
- A/B测试不同创意

### 3. 人群定向优化

| 定向类型 | 适用场景 |
|----------|----------|
| 基础定向 | 地域/年龄/性别 |
| 兴趣定向 | 偏好某类游戏的用户 |
| 行为定向 | 有过相关App使用行为 |
| 相似受众 | 种子用户扩展 |
| 再营销 | 流失用户召回 |

### 4. 投放节奏

| 阶段 | 策略 |
|------|------|
| **测试期** | 多素材、多人群、低预算测试 |
| **放量期** | 扩大量级、优化成本 |
| **稳定期** | 维持成本、精细化运营 |

## 四、分析框架模板

### 每日数据分析 checklist

```
□ 各渠道花费是否超预算
□ 各渠道CPI是否在目标范围内
□ 各渠道ROAS是否达标
□ 是否有异常数据（暴涨/暴跌）
□ 重点渠道转化漏斗是否正常
□ 素材表现排名
□ 今日优化动作记录
```

### 周度分析报告结构

```
1. 本周数据概览
   - 总花费、总新增、总收入
   - 整体CPI、ROAS、LTV

2. 渠道表现
   - 排名：量级、成本、回收
   - 本周 vs 上周对比

3. 素材表现
   - Top 3 素材
   - 淘汰低效素材

4. 问题诊断
   - 成本异常渠道
   - 转化漏斗异常

5. 下周计划
   - 优化方向
   - 预算调整
```

## 五、关键决策阈值

| 指标 | 警戒线 | 行动 |
|------|--------|------|
| CPI | 超过目标20% | 立即优化或暂停 |
| ROAS | < 100% | 暂停或大幅降预算 |
| CTR | < 1% | 必须换素材 |
| 7日LTV | < CPI | 暂停该渠道 |
| 次留 | < 30% | 检查产品/渠道匹配度 |
