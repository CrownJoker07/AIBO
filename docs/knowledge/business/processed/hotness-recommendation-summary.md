# 产品算法知识要点

> 来源: https://www.woshipm.com/pmd/723735.html
> 分类: business
> 主题: 产品经理需要了解的热度算法和个性化推荐算法

## 热度算法

### 核心公式
```
Score = S0 + S(Users) - S(Time)
```

### 三要素
| 要素 | 说明 |
|------|------|
| S0 初始热度 | 按内容类别差异化设置 |
| S(Users) 用户行为分 | click:1, favor:5, comment:10, share:20 |
| S(Time) 时间衰减 | 用指数函数模拟牛顿冷却定律 |

## 个性化推荐算法

### 基于内容的推荐
- 分词 → TF-IDF关键词权重 → 夹角余弦计算相似度
- 优点: 对用户规模要求低，适合冷启动

### 用户协同过滤
- 用户特征向量 → 欧几里得距离计算相似性
- 优点: 能发现用户潜在兴趣

## 产品经理要点

1. 产品初期用热度算法冷启动
2. 积累用户后引入个性化推荐
3. 实际产品常多算法结合使用
4. 需深入理解原理，结合产品特征优化
