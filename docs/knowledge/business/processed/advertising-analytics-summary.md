## 核心指标速查

### 成本指标
- **CPM**: 每千次展示成本
- **CPI**: 每安装成本 = 花费/新增安装
- **CPC**: 每次点击成本

### 效果指标
- **ROI**: 收入/投入×100%
- **ROAS**: 广告收入/广告花费×100%
- **LTV**: 用户生命周期价值

### 转化指标
- **CTR**: 点击率 = 点击/曝光
- **CVR**: 转化率 = 转化/点击

---

## 关键公式

```
ROAS > 100% → 广告盈利
LTV > CPI × 3 → 用户获取可持续
```

---

## 分析框架

### 渠道优先级
| 组合 | 行动 |
|------|------|
| 高量级 + 高回收 | 加投 |
| 高量级 + 低回收 | 优化素材/出价 |
| 低量级 + 高回收 | 扩展投放 |
| 低量级 + 低回收 | 暂停 |

### 优化优先级
1. 先看 ROAS 是否达标
2. 再看 CPI 是否合理
3. 最后看量级是否足够

---

## 决策阈值

| 指标 | 警戒线 | 行动 |
|------|--------|------|
| ROAS | < 100% | 暂停或降预算 |
| CPI | 超目标20% | 优化或暂停 |
| CTR | < 1% | 必须换素材 |
| 7日LTV | < CPI | 暂停渠道 |
| 次留 | < 30% | 检查匹配度 |
