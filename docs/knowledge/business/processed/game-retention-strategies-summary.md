# IAA小游戏留存率提升指南

> 业务知识 | 游戏运营 | 用户增长
> 来源：微信公众号文章

## 核心洞察

IAA小游戏面临严峻留存挑战，前10秒流失极高，次日留存通常不足10%。提升留存有六大关键策略。

## 六大留存策略

| 策略 | 核心要点 |
|------|----------|
| 开局不劝退 | 关注前X秒流失率，玩家是来放松不是受虐 |
| 奖励及时给 | 少数有意义行为立即给奖励，避免"画饼" |
| 社交互动 | 排行榜、分享机制增强粘性 |
| 适度活动 | 节假日限时活动，避免过度或缺失 |
| 持续更新 | IAA建议每周更新，别让玩家觉得游戏"挂了" |
| 次日奖励 | 必须给核心资源，如解锁新关卡/玩法 |

## 关键数据

- 解谜游戏次日登录奖励：留存 14% → 21%+
- 节假日活动：留存提升 20%
- 强制视频分享：在线时长提升约 30%

## 重要教训

**抽奖功能慎用**：
- 精准玩家更愿意硬刚，不精准玩家半点受不了挫折
- 抽到差奖直接流失，调整半年后选择放弃
