## 一、用户获取的核心挑战

### 市场现状

UA（用户获取）支出持续上升，每一美元都至关重要。传统的广告投放方式存在以下问题：

- **仅关注安装量**：无法衡量用户质量
- **数据孤岛**：广告数据与游戏内行为数据分离
- **归因模糊**：难以确定哪些渠道带来高价值用户
- **ROI 测算困难**：缺乏端到端的 ROAS 追踪

### 核心理念

> **猜测成本高昂，决策应基于数据而非直觉。**

AnalyticsIQ 帮助将广告活动直接与游戏内行为关联，实现真正的数据驱动 UA 策略。

## 二、AnalyticsIQ Acquisition 三大核心能力

### 1. 精准追踪广告影响

**功能描述：**
查看哪些广告、创意或广告网络带来安装，以及这些用户是否真正参与和付费。

**关键指标：**
| 指标 | 说明 |
|------|------|
| **安装来源** | 追踪每个安装的归因来源 |
| **参与度** | 安装后的用户参与行为 |
| **付费转化** | 从安装到付费的转化率 |

**应用场景：**
- 识别高效广告创意
- 评估不同广告网络质量
- 优化广告预算分配

### 2. 超越安装数据 - 漏斗分析

**功能描述：**
使用漏斗分析不同广告来源的玩家如何通过：
- 教程流程
- 早期关卡
- 变现步骤

**漏斗分析框架：**

```
广告曝光 → 安装 → 教程完成 → 首次游戏 → 首次付费 → 长期留存
    ↓         ↓         ↓           ↓          ↓          ↓
  CTR      IR     教程完成率    D1留存    付费转化率    LTV
```

**关键洞察：**
- 识别每个渠道的流失点
- 比较不同来源用户的质量
- 发现高价值用户特征

### 3. 测量真实 ROAS 和 ROI

**功能描述：**
使用队列分析比较不同来源的：
- 留存率（Retention）
- 生命周期价值（LTV）
- 变现能力（Monetization）

**ROAS 计算框架：**

```
ROAS = (LTV × 安装数) / 广告支出

按来源细分：
- ROAS by Network（按广告网络）
- ROAS by Creative（按广告创意）
- ROAS by Campaign（按广告活动）
```

**队列分析维度：**
| 维度 | 分析内容 |
|------|----------|
| **按渠道** | 不同归因渠道的用户价值对比 |
| **按时间** | D1/D7/D30 留存和 LTV 趋势 |
| **按行为** | 高价值用户的行为特征 |

## 四、UA 优化最佳实践

### 1. 渠道质量评估

**评估框架：**
```
高价值渠道特征：
✓ 高留存率（D1 > 40%, D7 > 20%）
✓ 高付费转化率
✓ 高 ARPPU
✓ 正向 ROAS

低价值渠道特征：
✗ 高安装量但低参与度
✗ 快速流失
✗ 低付费意愿
✗ 负向 ROAS
```

### 2. 创意优化策略

**A/B 测试框架：**
1. 定义测试假设
2. 创建不同创意版本
3. 追踪安装后行为
4. 分析长期价值
5. 迭代优化

**关键指标：**
- CTR（点击率）
- CVR（转化率）
- IPM（每千次展示安装数）
- 安装后参与度

### 3. 预算分配优化

**策略框架：**
```
阶段 1: 测试期
- 小预算测试多个渠道
- 收集数据建立基准

阶段 2: 优化期
- 暂停低效渠道
- 加大高效渠道投入

阶段 3: 扩展期
- 规模化高效渠道
- 持续监控 ROAS
```

## 六、案例研究：Midnite Games

**背景：**
- **知名游戏**：Slices（超过 6500 万下载）
- **团队规模**：10 人紧凑团队
- **挑战**：需要快速可靠的 UA 洞察，但团队资源有限

**使用方法：**
1. **漏斗分析**：追踪不同广告来源用户的行为路径
2. **队列分析**：比较不同渠道的留存和 LTV
3. **实时仪表板**：监控 UA 活动的即时效果

**成果：**
- 精确定位高价值用户来源
- 优化广告预算分配
- 提高 UA 整体 ROAS
