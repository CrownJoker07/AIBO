- 可扩展的视觉效果

- 视觉特效增强

#### 2. 扩展多平台覆盖 (Expand Multiplatform Reach)
- 移动平台优化体验

- 所有支持平台的多平台最新进展

#### 3. 解锁 Unity AI 可能性 (Unlock Possibilities with Unity AI)
- AI 驱动的代码生成工具
- 资产创建的 AI 辅助
- 运行时推理能力
- 集成在 Unity 编辑器和 Dashboard 中

- 更好的性能分析选项
- ProBuilder 改进

- UI Toolkit 更新

## 核心概念词汇

| 术语 | 说明 |
|------|------|
| Scene（场景） | 包含游戏环境和菜单，每个独特的场景文件相当于一个独特的关卡 |
| Avatar | 用于在不同骨骼之间重定向动画的接口 |
| Animation Clip | 动画数据，如"Idle"、"Walk"或"Run"等基本动作单元 |
| State Machine | Animator Controller 中的状态集合，包含状态之间的转换 |
| Sprite | 2D 图形对象，本质上是标准纹理 |
| Collider | 用于检测两个 GameObject 接触或重叠的组件 |
| Render Pipeline | 将场景内容显示到屏幕上的一系列操作 |
| Post-processing | 在图像显示前应用滤镜和效果来改善视觉效果 |
| Particle | 由粒子系统发射的小而简单的图像或网格 |

## 技术要点

### 渲染管线
Unity 6 支持多种渲染管线：
- Built-in Render Pipeline
- Universal Render Pipeline (URP)
- High Definition Render Pipeline (HDRP)
- 自定义渲染管线
