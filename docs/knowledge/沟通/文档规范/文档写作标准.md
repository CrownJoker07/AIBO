# 文档写作标准

> 统一文档格式，提升团队协作效率

---

## 文档结构

### 1. 标题

- 使用清晰的标题描述文档目的
- 包含日期和版本号（如需）

### 2. 概述

- 一段话说明文档背景和目的
- 目标读者是谁

### 3. 正文

- 使用标题层级组织内容
- 关键信息放在前面
- 适当使用表格和列表

### 4. 总结/行动项

- 提炼关键结论
- 列出后续行动和负责人

---

## 格式规范

- 标题层级：H1 > H2 > H3
- 中英文混排时使用中文标点
- 代码和命令使用 code formatting
- 链接使用有意义的描述文字

---

## 文档类型指南

| 类型 | 目标读者 | 长度 | 更新频率 |
|------|----------|------|----------|
| 需求文档 | 开发、设计 | 详尽 | 按需 |
| 技术方案 | 开发 | 详尽 | 按需 |
| 周报 | 管理者 | 精简 | 每周 |
| 会议纪要 | 参与者 | 中等 | 每次 |
| 知识库 | 全体成员 | 详尽 | 按需 |

---

## 文档 checklist

- [ ] 标题清晰，能独立说明内容
- [ ] 有概述帮助快速理解
- [ ] 结构清晰，便于浏览
- [ ] 无错别字和语法错误
- [ ] 链接和引用有效
