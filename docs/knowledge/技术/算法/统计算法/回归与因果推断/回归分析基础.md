# 回归分析基础

> 理解变量关系，预测业务指标

---

## 回归分析概述

回归分析用于研究自变量（X）与因变量（Y）之间的关系：

**Y = f(X) + ε**

- **线性回归**: Y = β₀ + β₁X + ε
- **逻辑回归**: 预测二分类结果概率
- **泊松回归**: 预测计数数据

---

## 线性回归

### 基本形式

```
Y = β₀ + β₁X₁ + β₂X₂ + ... + βₙXₙ + ε
```

### 关键指标

| 指标 | 含义 | 理想值 |
|------|------|--------|
| R² | 解释方差比例 | > 0.7 |
| Adj R² | 调整后 R² | 用于多变量比较 |
| p-value | 系数显著性 | < 0.05 |
| VIF | 多重共线性 | < 5 |

### 游戏行业应用

**用户留存预测**:
```
D7留存 = β₀ + β₁×D1时长 + β₂×新手引导完成 + β₃×首次付费 + ε
```

---

## 逻辑回归

### 适用场景

- 预测用户是否会付费
- 预测用户是否会流失
- 预测广告点击概率

### 输出解释

**Odds Ratio (优势比)**:
- OR = 1.5: 该因素使事件发生概率增加 50%
- OR = 0.8: 该因素使事件概率降低 20%

---

## 因果推断

### 相关 vs 因果

- **相关**: X 和 Y 同时变化，但不一定是因果关系
- **因果**: X 的变化直接导致 Y 的变化

### 因果推断方法

| 方法 | 适用场景 |
|------|----------|
| **随机实验 (RCT)** | 金标准，可直接推断因果 |
| **倾向得分匹配 (PSM)** | 处理观察数据的混杂 |
| **双重差分 (DID)** | 处理组 vs 对照组比较 |
| **工具变量 (IV)** | 处理内生性问题 |
| **断点回归 (RDD)** | 近似自然实验 |

### 游戏行业实例

**问题**: 付费用户留存更高，是因为付费本身还是因为这些用户本身更优质？

**分析方法**:
- 倾向得分匹配：匹配付费和未付费用户的基线特征
- 双重差分：对比付费前后留存变化

## 回归分析 checklist

- [ ] 确认变量关系方向
- [ ] 检查多重共线性（VIF < 5）
- [ ] 验证残差正态性
- [ ] 检查异常值影响
- [ ] 考虑非线性关系
- [ ] 区分统计显著与实际意义
- [ ] 避免过拟合

---

## 常见问题

1. **混杂变量**: 未控制的变量导致虚假相关
2. **多重共线性**: 自变量高度相关导致系数不稳定
3. **过拟合**: 在训练数据上表现好，泛化差
4. **因果推断错误**: 把相关当因果
