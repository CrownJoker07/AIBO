# 贝叶斯统计入门

> 用概率表达不确定性，持续更新认知

---

## 贝叶斯 vs 频率学派

| 视角 | 频率学派 | 贝叶斯学派 |
|------|----------|------------|
| 概率定义 | 长期频率 | 主观置信度 |
| 参数 | 固定未知 | 随机变量 |
| 信息来源 | 仅数据 | 数据 + 先验 |

---

## 贝叶斯定理

```
P(θ|data) = P(data|θ) × P(θ) / P(data)
```

| 符号 | 含义 |
|------|------|
| P(θ) | 先验概率 (Prior) |
| P(data\|θ) | 似然 (Likelihood) |
| P(θ\|data) | 后验概率 (Posterior) |
| P(data) | 边缘似然 (Evidence) |

---

## 核心概念

### 先验分布 (Prior)

在观察数据前对参数的认知：
- **无信息先验**: 均匀分布，如 Beta(1,1)
- **信息先验**: 基于历史知识，如 Beta(α,β)

### 后验分布 (Posterior)

结合数据后的更新认知：
```
Posterior ∝ Likelihood × Prior
```

### 共轭先验

| 似然分布 | 共轭先验 | 后验分布 |
|----------|----------|----------|
| 二项分布 | Beta | Beta |
| 泊松分布 | Gamma | Gamma |
| 正态分布 | Normal | Normal |

## 游戏行业应用

### 付费转化率估计

**场景**: 新功能上线，只有 100 个用户，10 人付费

**频率学派估计**: 10/100 = 10%

**贝叶斯估计**:
```
先验: Beta(2, 18)  # 假设历史均值 10%
后验: Beta(2+10, 18+90) = Beta(12, 108)
后验均值: 12/(12+108) = 10%
```

当数据少时，贝叶斯方法更稳健。

### 留存率预测

**多臂老虎机问题**:
- 同时测试多个方案
- 利用已有信息学习，同时探索新方案
- Thompson Sampling: 从后验分布采样选择方案

## MCMC 方法

当共轭先验不适用时，使用马尔可夫链蒙特卡洛：

| 方法 | 特点 |
|------|------|
| Gibbs Sampling | 条件分布采样 |
| Metropolis-Hastings | 接受-拒绝采样 |
| NUTS | No-U-Turn Sampler，高效 |

## 贝叶斯 A/B 测试

**优势**:
- 直接给出"方案 B 比 A 好的概率"
- 自然处理多重比较
- 可以提前停止实验（可信度足够时）

**实施步骤**:
1. 设定先验分布
2. 收集数据
3. 计算后验分布
4. 计算 P(B > A)
