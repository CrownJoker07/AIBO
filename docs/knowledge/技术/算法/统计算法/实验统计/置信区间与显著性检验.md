# 置信区间与显著性检验

> 理解统计结论的不确定性，避免错误解读数据

---

## 置信区间 (Confidence Interval)

### 定义

置信区间是对真实参数值范围的估计：
- **95% 置信区间**: 如果重复实验 100 次，95 次的区间会包含真实值

### 计算方法

**总体比例的置信区间**:
```
CI = p̂ ± z × √(p̂(1-p̂) / n)
```

其中 z = 1.96（95% 置信水平）

### 注意事项

- 置信区间不是"真实值有 95% 概率落在这里"
- 置信水平越高，区间越宽
- 样本量越大，区间越窄

## 显著性检验 (Hypothesis Testing)

### 基本概念

| 术语 | 定义 |
|------|------|
| **零假设 (H₀)** | 无效果/无差异的假设 |
| **备择假设 (H₁)** | 有效果/有差异的假设 |
| **p-value** | 在 H₀ 成立时，观察到当前结果的概率 |
| **显著性水平 (α)** | 拒绝 H₀ 的阈值，通常 0.05 |

### 决策规则

- p < α → 拒绝 H₀ → 结果显著
- p ≥ α → 无法拒绝 H₀ → 结果不显著

### 假设检验类型

| 类型 | 适用场景 | 示例 |
|------|----------|------|
| **单样本 t 检验** | 与已知值比较 | 留存率 vs 历史均值 |
| **双样本 t 检验** | 两组比较 | 实验组 vs 对照组 |
| **配对 t 检验** | 同一用户前后比较 | 优化前 vs 优化后 |
| **卡方检验** | 比例比较 | 付费率差异 |
| **Mann-Whitney U** | 非正态分布 | 收入分布比较 |

## 效应量 (Effect Size)

p 值只能说明"有没有差异"，效应量说明"差异有多大"。

| 效应量指标 | 计算 | 解释 |
|------------|------|------|
| Cohen's d | (μ₁ - μ₂) / σ | < 0.2 小, 0.5 中, 0.8 大 |
| Relative Difference | (B - A) / A | 百分比变化 |
| Odds Ratio | (p₁/(1-p₁)) / (p₂/(1-p₂)) | 优势比 |

## 游戏行业应用场景

### 留存率分析

```
D7 留存: 实验组 21% vs 对照组 18%
置信区间: [19.2%, 22.8%]
→ 我们有 95% 把握说真实提升在 19.2%-22.8%
```

### 付费转化分析

```
转化率提升显著吗？
- 样本量 > 10,000/组
- p = 0.002 < 0.05
- 结论: 差异显著
```

---

## 常见错误

1. **混淆统计显著与业务显著**: p < 0.05 但提升只有 0.1%
2. **忽视置信区间**: 只看 p 值不看区间宽度
3. **样本量不足**: 检测不到真实存在的差异
4. **多次检验**: 未校正多重比较
