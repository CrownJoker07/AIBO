## 核心文章概览

### 1. 社交系统设计师 (Social Systems Designer)

> 原文: [What is a Social Systems Designer?](https://lostgarden.com/2025/03/what-is-a-social-systems-designer/)
> 发布日期: 2025-03

**定义**：社交系统设计师定义、构建和平衡决定多玩家如何互动以产生期望社交体验的游戏机械系统。

**核心关注领域**：

| 领域 | 说明 |
|------|------|
| 社交物流 | 玩家何时、何地、如何互动？理想的社交密度是多少？ |
| 信任与社交协调 | 如何在从低信任到高信任的个人或群体间定制活动？ |
| 社交资本形成 | 游戏是否在玩家之间建立友谊和社交资本？ |
| 社交规范 | 游戏和更广泛社区的社交价值观和规范化行为是什么？ |
| 病毒性 | 社区如何通过引入新玩家来维持自身？ |
| 毒性生成 | 游戏是否主动产生不良玩家行为和毒性？ |
| 社交经济学 | 驱动期望社交行为的游戏经济平衡是什么？ |
| 社交可供性 | 塑造社交结构的游戏社区和用户流程的 UI/UX 方面是什么？ |

**为什么现在出现这个角色**：

1. **强烈的商业理由**：GaaS 团队发现朋友一起玩是留存的驱动力
2. **经过验证的最佳实践**：积累了需要深入研究和掌握的设计模型和最佳实践
3. **从业者社区**：积极分享和改进的社区

### 2. 善意游戏设计 (Kind Games)

> 原文: [Kind Games: Designing for Prosocial Multiplayer](https://lostgarden.com/2023/07/kind-games-designing-for-prosocial-multiplayer/)
> 发布日期: 2023-07
> 工作组成员: Daniel Cook, JC Lau, May Ling Tan, Joel Burgess, Tomo Moriwaki, Erin Drake Kajioka

**核心定义**：善意游戏是从一开始就设计有故意促进亲社会行为系统的多人游戏。

**善意价值观**：

| 价值观 | 说明 | 类型 |
|--------|------|------|
| 安全 | 玩家感到可以分享和贡献而不必担心恐惧 | 必须 |
| 相互依赖 | 玩家相互协调，共同实现比个人更多的成就 | 必须 |
| 共同目标 | 玩家为比自己更大的事情而努力 | 加速器 |
| 归属感 | 玩家属于一个社区，是基本的社交需求 | 加速器 |
| 同理心 | 玩家学会倾听和理解他人的观点 | 加速器 |
| 多样性 | 玩家重视多元身份和观点 | 维护 |
| 友谊形成 | 玩家与他人形成强烈、有意义的纽带 | 维护 |
| 健康冲突 | 不可避免的冲突以健康、富有成效的方式处理 | 维护 |

**典型案例**：
- Sky: Children of Light - 最佳实践地将新玩家引导进入健康友谊
- Journey - 早期成功的善意游戏示例
- Animal Crossing - 仔细管理多人互动以减少毒性
- Elden Ring - "Let Me Solo Her" 玩家展示了在残酷背景下的利他主义

### 3. 社交系统设计实践

> 原文: [A design practice for social systems](https://lostgarden.com/2023/04/a-design-practice-for-social-systems/)
> 发布日期: 2023-04
> 原始发表: ACM Journal Games: Research and Practice

**四个关键机会**：

1. **游戏服务 (GaaS) 与长期玩家繁荣**
   - 玩家留存是游戏服务的主要商业驱动力
   - 不健康的玩家最终会离开榨取性服务
   - 玩家健康和繁荣应成为顶级战略要求

2. **社交资本的重建**
   - 现代社会基础设施导致社交资本流失
   - 孤独的人渴望归属，可能依附于任何群体
   - 游戏可以促进人们相遇和保持联系

3. **设计更好的培养皿世界**
   - 虚拟空间允许快速改变基本的互动规则
   - 这是一个设计问题，可以迭代实现美学目标
   - 可以重新想象治理、裁决和立法的数字系统

4. **工具与价值观的统一**
   - 需要盘点可用工具和最佳实践
   - 必须思考这些工具应该用于什么目的
   - 社区最终建立在价值观之上

**社交系统设计学科愿景**：
- 应该是任何长期游戏的核心
- 应该专注于建立积极的社交资本
- 应该建立一套经过测试的设计实践
- 应该建立从业者的道德准则

## 设计模式摘要

### 亲社会设计模式

| 模式 | 说明 |
|------|------|
| 双赢互动 | 构建帮助他人变得容易且有回报的情境 |
| 前置承诺 | 明确声明游戏是合作空间，设定期望 |
| 牺牲 | 通过可见牺牲创造利他主义机会 |
| 对比突出 | 在残酷背景下突出善意行为 |
| 赠礼经济 | 模仿历史赠礼经济的激励机制 |
| 感知慷慨 | 增强利他意图的感知 |

### 反模式

| 反模式 | 说明 |
|--------|------|
| 有毒利他主义 | 付出过多导致倦怠 |
| 稀缺与丰富 | 稀缺导致毒性，过度丰富减少互动 |
| 自私 | 只关心自己利益 |
| 个人竞争 | 过度强调竞争激励自私 |
| 黑暗三联征 | 自恋、马基雅维利主义、反社会 |
| 恶意破坏 | 限制他人选择的敌意行为 |

## 商业价值

| 洞察 | 数据 |
|------|------|
| 留存至关重要 | GaaS 模式下，长期留存驱动商业成功 |
| 毒性导致流失 | 33% 玩家因骚扰退游；LoL 首次遭遇毒性的玩家流失率高 32 倍 |
| 友谊驱动留存 | 多个朋友一起玩时留存率飙升 |
| 社区增加有机获客 | 口碑仍是主要获客驱动力 |
| 品牌奠定基础 | 善意游戏是长期品牌建设策略 |

## 关键洞察

1. **社交系统设计是专业学科**：需要深入研究社交物流、信任、资本形成等领域
2. **善意游戏是商业机会**：朋友一起玩是留存驱动力，善意游戏同时满足道德和商业目标
3. **设计模式可学习**：双赢互动、牺牲、赠礼经济等模式可用于设计亲社会系统
4. **反模式需避免**：稀缺、自私、个人竞争、黑暗三联征行为会破坏社区
5. **过程是园艺**：开发者播种基础结构，社区培育社交规范
6. **衡量很重要**：需要超越留存和变现的指标来衡量善意
