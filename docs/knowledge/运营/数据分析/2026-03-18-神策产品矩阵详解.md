神策数据提供六大核心产品，形成完整的数据驱动业务闭环：

```
┌─────────────────────────────────────────────────────────────┐
│                      神策产品矩阵                            │
├─────────────────────────────────────────────────────────────┤
│  数据采集  │  数据分析  │  用户洞察  │  营销触达  │  效果验证  │
├─────────────────────────────────────────────────────────────┤
│  神策分析云  │  神策营销云  │  神策用户画像  │  神策智能运营  │

## 1. 神策分析云

### 产品定位

驱动业务的全场景数据分析平台

### 核心能力

| 能力 | 说明 |
|------|------|
| **十大分析模型** | 灵活组合、秒级响应 |
| **私有化部署** | 严格分级的权限体系，保障数据安全 |
| **行为洞察** | 探索不同业务中的关键行为，洞察指标背后的增长点 |
| **决策驱动** | 驱动业务决策 |
| **深度开发** | 支持深度开发，整合打通三方系统，灵活开放 |

### 十大分析模型（常见）

1. 事件分析
2. 漏斗分析
3. 留存分析
4. 分布分析
5. 用户路径
6. 归因分析
7. 用户分群
8. 用户画像
9. 间隔分析
10. 趋势分析

---

## 2. 神策营销云

### 产品定位

基于全域标签的数字化营销平台

### 核心能力

- 基于用户标签的精准营销
- 多渠道触达能力
- 自动化营销流程
- 营销效果追踪与优化

---

## 3. 神策用户画像

### 产品定位

帮助企业构建用户标签及画像体系

### 核心能力

| 能力 | 说明 |
|------|------|
| **标签体系构建** | 用户标签体系的系统化构建 |
| **自助管理** | 自助生产、维护、迭代用户标签 |
| **实时计算** | 标签人群实时计算、随需随取 |
| **全景画像** | 输出全景用户画像，利用标签形成用户分层 |
| **特征洞察** | 了解用户群体的特征属性和偏好概况，判断与预期是否一致 |

### 用户画像应用场景

1. **精准营销** - 基于用户特征的个性化触达
2. **产品优化** - 了解用户需求，指导产品迭代
3. **运营决策** - 数据驱动的运营策略制定
4. **用户分层** - 差异化服务和运营策略

---

## 4. 神策智能运营

### 产品定位

通过用户行为、属性、标签等数据筛选受众，实现自动化运营

### 核心能力

| 能力 | 说明 |
|------|------|
| **全局视角看板** | 把"宏观把控"和"微观处理"做到理想 |
| **自动化计划** | 自动化创建运营计划，又快又好的完成运营指标 |
| **降低时间成本** | 降低看数据提需求、等排期的运营时间成本 |
| **完整闭环** | 在一个平台上助力运营打造完整、持续迭代的闭环 |

### 智能运营流程

```
用户分群 → 策略配置 → 自动触达 → 效果追踪 → 策略优化
```

---

## 5. 神策广告分析

### 产品定位

多渠道广告效果分析与归因平台

### 核心能力

| 能力 | 说明 |
|------|------|
| **One ID 体系** | "一表"拉通全域用户数据 |
| **多渠道归因** | 多渠道广告转化归因，识别高价值传播路径 |
| **效果分析** | 支持主流渠道一站式效果分析，匹配最佳投放策略 |
| **用户洞察** | 支持多种分析模型助力广告投放、用户洞察 |

### 广告分析应用场景

1. **渠道效果评估** - 对比不同渠道的获客质量和成本
2. **归因分析** - 识别用户转化的关键触点
3. **投放优化** - 数据驱动的广告投放策略调整
4. **ROI 计算** - 准确计算广告投入产出比

---

## 6. 神策 A/B 测试

### 产品定位

全链路实验平台，支持多种试验类型

### 核心能力

| 能力 | 说明 |
|------|------|
| **四步创建** | 仅需四步即可完成试验创建 |
| **多种试验类型** | 编程试验、多链接试验、可视化试验等 |
| **实时报告** | 一键生成实时的专业测试报告，保证时效性与决策效率 |
| **全链路分析** | 打通神策分析模型，支持试验全链路深度下钻分析 |

### A/B 测试流程

```
定义假设 → 创建试验 → 流量分配 → 数据收集 → 结果分析 → 决策上线
```

### 支持的试验类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **编程试验** | 通过代码实现试验逻辑 | 复杂功能测试 |
| **多链接试验** | 不同URL版本的对比 | 落地页优化 |
| **可视化试验** | 无需代码的可视化编辑 | UI/文案优化 |

## 产品协同关系

### 数据流转闭环

```

│                        数据采集层                           │
│         全端数据采集 → One ID 打通 → 实时同步               │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        数据分析层                           │

└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        用户洞察层                           │

└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        营销触达层                           │
│         神策营销云 + 神策智能运营（自动化）                  │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        效果验证层                           │
