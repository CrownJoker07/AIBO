## 核心框架

### 玩家旅程七阶段模型
```
获取 → 引导(FTUE) → 初期参与 → 首次购买 → 持续参与 → 长期留存 → 再参与
```

每个阶段相互依赖，持续互相通知和改进。玩家不应察觉阶段过渡，体验应自然流畅。

## 用户获取

### ASO vs 付费UA

| 维度 | ASO | 付费UA |
|------|-----|--------|
| **速度** | 慢（数周/数月） | 快（几小时） |
| **控制力** | 低（依赖算法） | 高（完全可控） |
| **可衡量性** | 有限 | 精确到分 |
| **定位精度** | 广泛搜索 | 精准人群 |

### 最佳实践
- **从付费UA开始，用ASO支持**
- 付费UA解决速度和规模问题
- ASO解决效率和长期可见性问题
- 两者结合创造飞轮效应

### 饱和式投放案例（Royal Kingdom）
- 全球发布前：10,000+广告创意
- 峰值：每日500+新创意
- 覆盖28+国家
- 明星代言创造文化相关性

## 变现策略

### 角色收集（#1元游戏机制）
- 70%顶级移动游戏采用
- 88%中核游戏使用

#### 三大心理驱动
1. **控制感与完整性** - 收集创造秩序，满足成就感
2. **炫耀心理** - 稀有角色是技能和奉献的象征
3. **身份表达** - 角色选择反映个人风格

#### 变现机会
- 游戏内商店直接购买
- 独家角色（仅限付费）
- 升级资源包
- 活动限定角色

### 外观变现

#### 核心原则
- **不影响游戏平衡**
- 仅专注于美学和个性化
- 85%美国玩家认知外观道具
- 81%愿意付费

#### 十大策略
1. 了解受众偏好
2. 平衡独占性与可及性
3. 提供多样化选择
4. 质量优于数量
5. 整合到游戏玩法中
6. 有效推广展示
7. 战略性定价
8. 持续更新商店
9. 培养社区感
10. 倾听玩家反馈

## 留存与参与

### 关键留存数据（Royal Kingdom案例）
| 指标 | 数据 |
|------|------|
| Day 1 留存 | 38.3% |
| Day 7 留存 | 19.2% |
| Day 30 留存 | 9.5% |
| DAU/MAU | 32.8% |
| 平均会话时长 | 16.7分钟 |
| 每日会话数 | 4.35次 |

### 留存驱动因素
1. **每日任务** - 明确的每日目标
2. **进度系统** - 升级、技能树、叙事进度
3. **社交功能** - 公会、好友、多人模式
4. **内容更新** - 稳定的双周更新节奏
5. **连胜机制** - 不想中断的心理

### 再参与策略
- 流失玩家更容易重新参与（已熟悉游戏）
- "我们想念你"活动 + 独特激励
- 基于玩家行为定制再参与路径
- 重新点燃兴趣比获取新用户成本更低

## 首次购买转化

### 关键原则
- 玩家对过度激进变现非常敏感
- IAP和广告应自然融入，感觉不突兀
- 让花钱感觉像游戏的自然延伸

### 有效策略
| 策略 | 说明 |
|------|------|
| **新手礼包** | 高价值的限时优惠 |
| **限时促销** | 创造紧迫感 |
| **差一点成功** | 差1-2步完成时提供小额帮助 |
| **连胜保护** | $0.99续命保连胜 |

## 案例研究：Royal Kingdom

### 成就概览
- 5500万+下载
- $1.45亿+收入（不到一年）
- 无强制广告，完全依赖IAP

### 成功公式
```
精良产品 + 智能变现 + 强力留存 + 激进营销 = 爆款
```

### 关键经验
1. **软启动至关重要** - 1年测试期确保产品成熟
2. **内容管线** - 稳定的双周更新是留存锚点
3. **视觉品质** - 精良制作创造高端感
4. **小额高频** - $0.99续命比大礼包更有效
5. **规模测试** - 同时测试大量广告创意

## 参与度机制设计

### 什么是参与度机制？
专为强化用户行为和提高留存而设计的产品功能，包括：连胜系统、进度系统、应用内货币、社交问责功能、LiveOps活动。

### 参与度机制支持增长的原理
- 提高留存率
- 强化每日习惯
- 增强变现循环
- 用户感受到进步、问责或动力时更可能回归

### 成功案例速查

| 应用 | 类型 | 核心机制 | 增长策略 |
|------|------|----------|----------|
| **AirBuds** | 社交音乐 | 实时分享、每周回顾 | 社交优先设计 → 网络效应 |
| **FlareFlow** | 短剧 | 每日任务、登录奖励 | 复制已验证策略到新品类 |
| **Learna** | AI辅导 | 连胜系统、进度追踪 | 成熟机制 + 新技术 |
| **Pengu** | 社交宠物 | 共同抚养、双货币、LiveOps | 社交问责 × 游戏化 |

### 核心启示
> **突破性增长往往来自将成熟的参与度机制应用到新的品类，而非发明全新的机制。**

## 移动广告创意趋势（2025）

### 互动广告成为标配
- 非游戏应用互动广告支出增长 **+20%**
- 顶级游戏可玩广告支出领先 **+35%**
- 可玩广告ITI转化率提升 **16x**

### UGC内容效果
- 引入UGC后ITI转化率提升 **+152%**
- UGC + 应用UI演示 = 高效组合
- 展示应用UI让用户提前开始onboarding

### 生成式AI应用
- 静态→动态（添加动作）
- 视频→可玩（规模化测试变体）
- UGC本地化（翻译、配音、口型同步）
