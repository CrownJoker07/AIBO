## 核心方法论

### 游戏公司评估五力模型

```
┌─────────────────────────────────────────────────────────────┐
│                    Naavik 评估框架                           │
├─────────────────────────────────────────────────────────────┤
│  产品力     │  游戏质量、品类定位、创新能力、IP价值          │
├─────────────────────────────────────────────────────────────┤
│  运营力     │  LiveOps能力、内容更新节奏、社区管理           │
├─────────────────────────────────────────────────────────────┤
│  技术力     │  引擎技术、服务端架构、数据分析能力            │
├─────────────────────────────────────────────────────────────┤
│  商业力     │  变现效率、用户获取ROI、收入多元化             │
├─────────────────────────────────────────────────────────────┤
│  组织力     │  团队能力、公司文化、人才储备                  │
└─────────────────────────────────────────────────────────────┘
```

### 关键指标体系

| 类别 | 核心指标 |
|------|----------|
| **增长** | DAU/MAU、下载量、自然/付费比例 |
| **留存** | D1/D7/D30留存、DAU/MAU比率 |
| **变现** | ARPU、ARPPU、付费率、LTV |
| **效率** | CAC、ROAS、回本周期 |
